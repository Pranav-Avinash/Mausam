# Mausam

A simple and elegant weather application built with React and Vite.

## Live Demo
Check out the live application here: [Mausam](https://Pranav-Avinash.github.io/Mausam/)

## Features
- Real-time weather updates
- Search functionality for different locations
- Responsive design
- Displays various weather conditions (Clear, Cloudy, Rain, Snow, etc.)

## Technologies Used
- React
- Vite
- CSS

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Build for production: `npm run build`
